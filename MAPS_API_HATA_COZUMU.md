# Maps JavaScript API Etkinleştirme

## Hata
```
ApiNotActivatedMapError - Maps JavaScript API aktif değil
```

## Çözüm Adımları

1. **Google Cloud Console'a gidin:**
   https://console.cloud.google.com/apis/library

2. **Arama kutusuna yazın:** Maps JavaScript API

3. **"Maps JavaScript API" kartına tıklayın**

4. **"ENABLE" (ETKİNLEŞTİR) butonuna tıklayın**

5. **2-3 dakika bekleyin** (API'nin aktif olması biraz zaman alır)

6. **Sayfayı yenileyin** ve tekrar arama yapın

## Etkinleştirmeniz Gereken API'ler

✅ Geocoding API (zaten aktif)
✅ Places API (zaten aktif)  
❌ **Maps JavaScript API** ← BUNU ETKİNLEŞTİRİN

## Not
Maps JavaScript API olmadan harita görüntülenemez, sadece arka planda arama yapılır.
